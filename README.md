# Proyecto de Ventana Modal Simple

Este es un proyecto sencillo para demostrar cómo crear una ventana modal (o pop-up) utilizando HTML, CSS y JavaScript puro.

## Características

*   **Abrir Modal:** Haz clic en el botón para mostrar la ventana modal.
*   **Cerrar Modal:** Cierra la ventana modal haciendo clic en el botón de "exit".
*   **Cerrar con Overlay:** Cierra la ventana modal haciendo clic en cualquier parte del fondo oscuro fuera de la ventana.

## Tecnologías Utilizadas

*   **HTML5:** Para la estructura básica de la página.
*   **CSS3:** Para los estilos, incluyendo el diseño del modal y el uso de Flexbox para el centrado.
*   **JavaScript (ES6):** Para la lógica de abrir y cerrar la ventana modal.

## Cómo Usarlo

1.  Clona o descarga este repositorio.
2.  Abre el archivo `index.html` en tu navegador web preferido.

¡Y eso es todo! No se requiere instalación ni dependencias.
